Operating system related information - [click to expand]
Windows (XP/Vista/7/8/8.1)
Software is fully compatible.
Linux
1. You need to add yourself to "dialout" group to allow chrome/chromium serial port access.
FIX: You can do so by executing: sudo usermod -aG dialout YOUR_USERNAME, relog or restart after you do so.

2. Current implementation of chrome.serial API contains a "broken" non-raw configuration for opened ports, that leads to complete Serial malfunction.
Fix will reach stable release of Google Chrome/Chromium in version 32 (scheduled to be released in January).

WORKAROUND: After you connect serial adapter to the computer you need to apply raw settings to the serial port, for example: stty -F /dev/ttyACM0 raw
Auto-connect should be manually disabled until this issue gets resolved.

Deluxe TX related information - [click to expand]
Deluxe TX is based on Atmega32u4 chip, which have usb support directly on the chip (no serial to usb dongle required).

Because of this hardware "setup" configurator requires optional USB permissions to restart the module before connecting.
This means, that you won't be able to connect to this module unless you allow configurator to access USB devices.
