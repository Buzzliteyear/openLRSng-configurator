Note

All settings are transfered through the TX module to RX module via wireless connection.
You shouldn't connect RX module to the configurator unless you need to flash it with new firmware.
You don't need to connect the RX module to the computer, giving power to the RX module (from battery or other power source) is enough.

Receiver tries to connect on startup for 0.5 seconds, if this fails try bridging Port-1 and Port-2 on your receiver with a jumper.

Please wait for the TX module to establish connection with the RX module.
